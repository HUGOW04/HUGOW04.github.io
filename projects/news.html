Automated News Fetcher

An automated bot that fetches and filters relevant news articles, delivering real-time updates via a Discord webhook.

Project Overview

This project is an automated bot designed to fetch and filter interesting news articles from multiple sources. It delivers real-time updates through a Discord webhook, ensuring that users receive only relevant and engaging content.

Functionality

The system runs 24/7, continuously monitoring news sources. Alerts are sent at 12 AM each day, summarizing the latest articles. Users can scroll through the headlines and click on links to read the full articles if desired.

Technical Details

The bot uses web scraping and filtering logic to ensure only high-quality news is shared. Integration with Discord allows for seamless delivery of notifications and summaries, providing an efficient way to stay informed without being overwhelmed by information.
