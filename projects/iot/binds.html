Automated Blinds with WiFi and NTP

This project involves an automated blind system controlled via WiFi. The system uses an Arduino with a stepper motor to control the movement of the blinds. It is programmed to move the blinds up or down based on the time of day and day of the week, using an NTP client to synchronize time.

Key Features:

  • WiFi Connectivity: Allows remote control and status monitoring through a web interface.
  • Stepper Motor Control: Precisely moves the blinds up or down based on programmed instructions.
  • NTP Time Synchronization: Ensures accurate timing for automatic blind movements.
  • EEPROM Storage: Saves the state of the blinds to retain information across reboots.

How It Works:

The system connects to your WiFi network and uses an NTP client to get the current time. Based on the time of day and the day of the week, it will automatically move the blinds up or down. For instance, the blinds may be set to open at 6 AM on weekdays and close at 10 PM. You can also manually control the blinds through a web interface provided by the embedded server on the Arduino.

The code utilizes EEPROM to store the last state of the blinds, ensuring that the system resumes in the correct state even after a power cycle.
