x86 Operating System

This x86 operating system is crafted using Assembly language with a 16-bit boot mode. It transitions to a 32-bit mode for enhanced functionality and performance. The OS features integrated I/O operations allowing keyboard input, CPU management, and screen clearing.

It includes custom memory management with a malloc implementation developed in C, without relying on external libraries. The system is built with both Assembly and C, showcasing my ability to develop low-level system software with custom libraries.

Key Features:

  • 16-bit Boot Mode: Initial boot phase in 16-bit mode for compatibility and setup.
  • 32-bit Mode Transition: Switches to 32-bit mode for advanced operations and better performance.
  • Custom I/O Operations: Supports keyboard input, CPU management, and screen clearing.
  • Custom Memory Management: Implemented malloc in C without external libraries.
  • Assembly and C Integration: Combines Assembly for low-level control with C for higher-level functionality.
