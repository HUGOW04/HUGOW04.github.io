CS:GO Aimbot

CS:GO Aimbot

This project involves the development of a custom aimbot for Counter-Strike: Global Offensive (CS:GO). The aimbot calculates the vector from the player's gun to the enemy players by reading memory values and precisely adjusts the crosshair to aim at the enemy when the gun is fired.

The aimbot integrates deeply with the game's memory to ensure accuracy and speed, giving the player a significant edge in targeting enemies. It demonstrates advanced memory manipulation, vector calculations, and real-time input handling.

Key Features:

  • Memory Reading: Accesses and reads enemy player positions directly from the game's memory.
  • Vector Calculation: Calculates the precise vector from the player's gun to the target for accurate aiming.
  • Real-time Input Handling: Moves the cursor in real-time to align with the calculated aim vector when firing.
  • High Accuracy: Ensures precise aiming for effective targeting in competitive gameplay.

Custom Cheat Engine

Memory Manipulation Tool

This project features the development of a custom Cheat Engine designed for reading and writing memory within running processes. The tool is capable of scanning a process by its Process ID (PID) to search for specific memory addresses and allows users to modify those memory values directly. This enables various manipulations, such as altering in-game resources, changing player stats, or bypassing restrictions.

The engine is built to interact with system memory at a low level, providing powerful and precise control over the target process. It is useful for both educational purposes in understanding memory management and for creating custom mods or cheats in gaming environments.

Key Features:

  • Process Scanning: Scans processes using their Process ID (PID) to locate specific memory addresses.
  • Memory Reading and Writing: Reads current values from memory and allows users to write new values to those addresses.
  • Memory Search: Searches for specific memory addresses based on value, type, and other parameters.
  • Real-Time Modifications: Enables real-time changes to running processes without requiring a restart.
  • Precision Control: Provides detailed access to memory, enabling specific and accurate adjustments.

Hex Editor

Custom Hex Editor

This Hex Editor is a custom-built tool designed for viewing and editing the raw hexadecimal data of files. It allows users to open binary files and directly manipulate their content at the byte level, providing a powerful way to modify file structures, debug applications, or explore file formats.

The editor features an intuitive interface that displays both the hexadecimal and ASCII representations of file data, making it easy to navigate and understand the structure of complex files. It supports searching for specific byte patterns, editing values in real-time, and saving changes directly to the file.

Key Features:

  • Hexadecimal and ASCII Display: Provides a dual view of file data in both hex and ASCII formats for easier understanding and editing.
  • Byte-Level Editing: Allows precise modifications at the byte level, enabling detailed control over file contents.
  • Pattern Search: Includes search functionality to locate specific byte patterns within the file for quick navigation.
  • Real-Time Editing: Supports live editing with immediate updates to the file, reflecting changes as they are made.
  • File Integrity Management: Ensures that file structure remains intact during editing, with features to prevent corruption.

Input Overlay Display

Keyboard and Mouse Overlay

The Input Overlay Display project is a tool designed to visualize keyboard and mouse inputs on screen in real-time. This tool captures user inputs and displays them as a transparent overlay, showing the exact keys pressed and mouse movements as they occur. It is particularly useful for gaming streams, tutorials, and presentations where it's important to demonstrate precise input actions.

The overlay is customizable, allowing users to adjust the appearance, size, and position on the screen. It supports multiple input devices and displays inputs in a clear and intuitive format, making it easier for viewers to follow along with the actions being performed.

Key Features:

  • Real-Time Input Display: Captures and displays keyboard and mouse inputs in real-time as a visual overlay on the screen.
  • Customizable Overlay: Allows users to customize the appearance, including colors, size, and position of the overlay.
  • Multi-Device Support: Supports various input devices, including full keyboards, gaming mice, and controllers.
  • Transparency and Layout Options: Provides options for transparency and layout to ensure the overlay complements the background content.
  • Streaming and Tutorial Use: Ideal for live streaming, video tutorials, and presentations to clearly show user inputs.
