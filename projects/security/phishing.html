Phishing Website

This C++ application simulates a phishing website designed to capture login credentials from unsuspecting users. The server hosts a fake login page that mimics legitimate sites to trick users into entering their usernames and passwords. The captured credentials are then sent to the attacker for unauthorized access.

Key Features:

  • Fake Login Page: Presents a login form that looks like a legitimate website's login page.
  • HTTP Server: Hosts the login page and handles incoming HTTP requests.
  • Credential Capture: Uses JavaScript to send login credentials to the attacker's server.
  • Static File Serving: Serves the phishing page and associated files from a specified directory.
  • Educational Use: Demonstrates phishing techniques for security training and awareness.

How It Works:

The application sets up a basic HTTP server that serves a login page from a predefined directory. The HTML page contains a form where users enter their credentials. When the form is submitted, JavaScript sends the entered data to a remote server controlled by the attacker. This setup is used for educational purposes to show how phishing attacks work and to help in developing defenses against them.

Installation and Usage:

Compile the C++ source code in a Windows environment and run the executable to start the phishing server. Place the phishing HTML page and associated files in the specified directory. Open the server in a browser and simulate user login to capture credentials. Use this setup responsibly and only in a controlled, educational environment.
