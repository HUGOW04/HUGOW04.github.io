Trojan Horse

This C++ application is a Trojan horse designed to execute a variety of malicious activities on a Windows system. It includes features for remote control, file and system manipulation, data collection, and more. The Trojan horse can connect to a command-line interface, manage files, capture user data, and disrupt system operations.

Key Features:

  • Command Line Access: Connects to a command-line interface for executing system commands remotely.
  • File and Folder Management: Allows for changing directories, removing files and folders, and encrypting files.
  • System Information Retrieval: Gathers detailed information about the system's hardware and software.
  • Process Management: Provides functionality to kill running processes on the target machine.
  • Keylogging: Captures and logs keystrokes to gather sensitive information like passwords.
  • Error Display: Displays custom error messages on the target's screen to mislead or confuse the user.
  • Screenshots: Takes screenshots of the user's desktop to monitor activity.
  • Browser Data Extraction: Extracts data from Google Chrome, including saved passwords and browsing history.
  • Webcam Access: Captures images from the system's webcam to gather visual information.

How It Works:

Upon execution, the Trojan horse hides the console window to avoid detection. It connects to a remote server or command center to receive and execute commands. The Trojan performs a variety of actions including managing files, killing processes, capturing keystrokes, and displaying error messages. It also takes screenshots and extracts sensitive data from the browser.

Installation and Usage:

Compile the C++ source code in a Windows environment and run the executable to start the Trojan horse.
