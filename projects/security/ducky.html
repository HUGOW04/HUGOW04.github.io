Rubber Ducky

Rubber Ducky USB

This project involves a Rubber Ducky USB device capable of executing automated keystroke injections on a Windows system. It is designed to demonstrate how a simple USB device can be used to deliver malicious payloads by simulating keyboard input. The Rubber Ducky can execute commands rapidly, allowing for actions such as retrieving system information, manipulating files, and gaining unauthorized access to systems. The project aims to educate users about the security risks associated with USB-based attacks and the importance of proper safeguards against such threats.
