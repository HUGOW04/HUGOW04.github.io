Rubber Ducky

Rubber Ducky USB

This project involves a Rubber Ducky USB device capable of executing automated keystroke injections on a Windows system. Unlike traditional BadUSB devices, this version works with an SD card USB without requiring additional software. It demonstrates how a simple USB device can be used to deliver payloads by simulating keyboard input. The Rubber Ducky can execute commands rapidly, enabling actions such as retrieving system information, manipulating files, and gaining unauthorized access.

The goal of this project is to raise awareness about USB-based security threats and emphasize the need for proper safeguards.
