Blue Screen of Death (BSoD)

This C++ application demonstrates how to trigger a Blue Screen of Death (BSoD) on a Windows system. By using low-level NT API functions, the application simulates a critical system error that causes the operating system to crash and display a BSoD.

Key Features:

  • Privilege Adjustment: Elevates process privileges to allow system-level shutdown operations.
  • BSOD Simulation: Uses NT API functions to simulate a critical error that triggers a BSoD.
  • Windows API Integration: Utilizes Windows NT API for manipulating system privileges and errors.
  • Educational Use: Demonstrates system-level operations and the impact of critical system errors.

How It Works:

The application first elevates its privileges to allow system-level operations. It then invokes a critical system error using the `NtRaiseHardError` function with a specific error status that causes the operating system to crash and display a BSoD. This is used to demonstrate the effects of critical system errors.

Installation and Usage:

Compile the C++ source code in a Windows environment with appropriate permissions. Run the executable to trigger a BSoD. Use this application responsibly and only in a controlled environment, as it will crash the system and may result in data loss.
