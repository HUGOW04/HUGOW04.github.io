Ransomware

This C++ ransomware program is designed to demonstrate the functionality of file encryption and decryption on a Windows system, encrypting user files and demanding a ransom for decryption. The program uses a custom encryption algorithm to lock files, with a corresponding method to decrypt them. It is intended strictly for educational purposes, showcasing the risks of ransomware and helping users understand how such threats operate.

The application focuses on encrypting user files while avoiding critical system directories to prevent damage to the operating system. It reports progress throughout the encryption and decryption process, ensuring users are informed of the status. By integrating the Windows API, it efficiently handles file operations and interacts with the system to access directories and manipulate files.

The ransomware works by encrypting files in user directories, modifying them in blocks, and appending a .tu extension to indicate they are encrypted. To decrypt the files, a decryption tool is used along with the key that reverses the encryption process, restoring files to their original form. System-critical files are excluded from encryption to ensure the operating system remains functional.
