AntiVirus

Antivirus is a cross-platform application developed using C++ and OpenGL for rendering, ensuring compatibility across Windows, macOS, and Linux. It is designed to detect and manage malware threats efficiently, with both file-specific and full system scanning features. The application uses SHA-256 hashing for comparing files against known malware signatures, offering enhanced security and modern cryptographic methods. The OpenGL-rendered user interface ensures a consistent and responsive experience across all platforms.

Antivirus allows users to scan individual files or perform full system scans, recursively analyzing files within a specified directory. Real-time status updates keep users informed about the scanning progress, and results are logged to a text file for easy access and review.

The application leverages SHA-256 hashing, which computes a unique hash for each scanned file, compared against a database of known malware hashes. This secure method ensures reliable malware detection. For full system scans, the same process is applied across all files in the target directory, providing real-time updates on the results.
