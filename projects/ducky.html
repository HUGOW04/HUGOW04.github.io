Rubber Ducky USB Device

A Rubber Ducky USB device capable of executing automated keystroke injections on Windows systems using an SD card, without requiring additional software.

Project Overview

This project demonstrates a Rubber Ducky USB device that can simulate keyboard input to execute payloads automatically on a Windows system. Unlike traditional BadUSB devices, this version works with an SD card USB and does not require any additional software installation.

Functionality

The Rubber Ducky can rapidly execute commands, enabling tasks such as retrieving system information, manipulating files, or gaining unauthorized access. It showcases how a simple USB device can become a powerful security threat.

Educational Purpose

The goal of this project is to raise awareness about USB-based security threats, demonstrating the importance of proper safeguards, user vigilance, and security best practices when using external devices.
